Get a token

  1. ApplicationKey von https://trello.com/1/appKey/generate kopieren
  2. https://trello.com/1/authorize?key=SUBSTITUTEWITHYOUAPPLICATIONKEY&name=My+Application&expiration=never&response_type=token&scope=read,write besuchen
  3. Runtergeladene Datei öffnen
  4. Token kopieren

Siehe auch: https://trello.com/docs/gettingstarted/index.html#getting-a-token-from-a-user
